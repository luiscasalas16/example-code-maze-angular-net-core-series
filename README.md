# code-maze-angular-net-core-series

Proyecto de ejemplo de frontend en Angular y backend en .Net Core. El código es el resultado de realizar los tutoriales de las siguientes referencias:
- (https://code-maze.com/net-core-series/)
- (https://code-maze.com/angular-series/)

Se ajustó la aplicación para que utilizara base de datos MSSQL.

Para ejecutar el ejemplo se debe:
- Generar la base de datos con el archivo "database.sql".
- Actualizar el connection strings correspondiente en el archivo "brackend\AccountOwnerServer\appsettings.json".
- Ejecutar la aplicación de backend.
- Ejecutar la aplicación de frontend.
